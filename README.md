# SwitcherIO HomeAssistant

스위쳐 아이오 Switch를 홈 어시스턴트에 추가 합니다.
HACS에서 Custom Integration 추가하신담에 여기 경로 추가하세요.
블루투스 모듈이 필요합니다. 링커는 혼자 쓰고 있는데 나중에 추가 할 계획입니다.

```yaml
switch:
  - platform: switcher_io
    mac: 'XX:XX:XX:XX:XX:XX'

```
